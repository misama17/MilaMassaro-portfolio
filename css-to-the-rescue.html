CSS to the rescue

Minor Webdesign & Development

Tijdens deze vierweekse cursus heb ik geëxperimenteerd met CSS-technieken die nieuw voor mij waren. Het doel was om de mogelijkheden van moderne CSS verder te verkennen en een innovatieve, interactieve ervaring te creëren met uitsluitend vanilla HTML en CSS. Frameworks, preprocessors, libraries en JavaScript waren hierbij niet toegestaan, waardoor de focus volledig lag op de kracht van CSS zelf.

Gedurende het project heb ik ontdekt hoe uitgebreid CSS is en hoe veel je ermee kunt bereiken. Veel functies bieden verrassend veel mogelijkheden voor interactie, animatie en lay-out. Door bewust buiten mijn comfortzone te stappen, heb ik verschillende technieken onderzocht en toegepast die ik voorheen nog niet gebruikte.

Een belangrijk uitgangspunt van deze cursus was het vermijden van classes en IDs. Hierdoor werd ik gestimuleerd om creatiever gebruik te maken van de vele beschikbare CSS-selectors en de structuur van HTML beter te benutten. Daarnaast moest ik oplossingen bedenken zonder JavaScript, wat mij dwong om anders naar interactie en functionaliteit te kijken.
